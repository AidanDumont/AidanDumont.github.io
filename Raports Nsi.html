Apprentissage du langage html et CSS

Pendant ces vacances j'ai presonnellemnt beaucoup appris sur les nouveaux languages html et CSS sur lesquels j'étaient inexpérimentés.

En effet j'ai appris:

  • De quoi est constitué un élement html

  • Comment est un composé un fichier html. En effet il est composé de 3 balises ouvrantes et fermantes respectivement appelé html,head et body

    html est l'élément racine qui entoure le contenue de la page

    head est l'élément contenant toutes les choses qui font partie de la page HTML mais qui ne sont pas du contenu affiché.(ex:liens,...)

    body est l'élément qui contient toout ce que que l'utilisateur voudrait afficher (ex:images, textes...)

  • Comment est structurré un fichier CSS
  • Comment insérer des liens pour intégrer un élément CSS ou un autre élément html pour ainsi créer une nouvelle page
  • et enfin créer un fichier CSS pour modifier l'aspect des fichier html (marge,couleur...)
